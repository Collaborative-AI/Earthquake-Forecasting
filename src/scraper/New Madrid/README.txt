README file for NewMadridScraper.py

Summary:
Converts a TXT file from the New Madrid Earthquake Catalog into a CSV file.
The data can be found at http://folkworm.ceri.memphis.edu/catalogs/scratch/cat_s1072011_4222.

Label Abbreviations:
The label abbreviations are explained on the catalog's website and is linked here:
http://folkworm.ceri.memphis.edu/catalogs/html/cat_nm_help.html

The following lines can be modified:
Lines 12-14:    Modify the header of the CSV file
Line 32:        Modify the file path of the input .txt file
Line 33:        Modify the file path of the output .csv file
