README file for SRCMODScraper.py

Summary:
Scrapes all events on the Earthquake Source Model Database website accessible with "http://equake-rc.info/SRCMOD/searchmodels/allevents/". The data is stored in a CSV file generated in a new folder as SRCMOD/srcmod.csv.

The following lines can be modified:
Line 14:		Modify the location and name of the CSV file that is generated.
Lines 18-20:	Modify the header of the CSV file
Line 35:		Modify the URL that the helper function should scrape. By default, set
			to the SRCMOD website linked before.
