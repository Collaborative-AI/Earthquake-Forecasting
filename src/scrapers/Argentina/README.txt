README file for ArgentinaScraper.py

Summary:
Converts an XML file of earthquakes in the NW Argentine Andean Foreland into
a CSV file including time and magnitude measurements.
The XML file can be found at: https://doi.org/10.31905/YTIR1IED

The following lines can be modified:
Line 38:        Modify the file path of the input .txt file
Line 39:        Modify the file path of the output .csv file
