README file for Turkey Earthquake Catalog (1915-2021)

Source Link:
https://www.kaggle.com/datasets/atasaygin/turkey-earthquakes-19152021

Summary:
CSV file containing earthquakes from 1915-2021 AD that "occurred in Turkey and the close coverage area," as explained by the Kaggle source. Also, the dataset is owned by "Boğaziçi Üniversitesi Rektörlüğü", and "it can only be used for uncommercial issues with regards to "Boğaziçi Üniversitesi Kandilli Rasathanesi ve Deprem Araştırma Enstitüsü Bölgesel Deprem-Tsunami İzleme ve Değerlendirme Merkezi".

The CSV features a Turkish header, and the translates are included in the Kaggle source. The cleaned CSV features key data points in English.
